========================================
KEY REMAPPER - QUICK START
========================================

HOW TO USE:
-----------

Just double-click "RUN.bat" and wait!

The first time you run it:
- It will automatically install everything needed
- Build the program (takes 1-2 minutes)
- Then launch it

After that:
- It will launch instantly (no building needed)

REQUIREMENTS:
-------------
- Windows computer
- Internet connection (first time only, to download dependencies)
- Python installed (if not installed, the program will tell you)

WHAT THIS DOES:
---------------
This program lets you remap keyboard keys to randomly output other keys.

Example: Make "A" randomly type "W" or "E" instead.

See USER_INSTRUCTIONS.txt for how to use the program once it's running.
